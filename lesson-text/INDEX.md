# 🚀 2人でつくる！ミニゲーム・ポータルサイト開発講座

この講座では、協力して4つのミニゲームが遊べる「ポータルサイト」を作ります。
プロのエンジニアと同じように、**Git（ギット）**を使ってお互いのプログラムを合体させる体験をしましょう！

2. [【実践1】じゃんけんゲームを作ろう（Ruby）](./02_janken_ruby.md)

4. [【チーム】Gitで合体＆コンフリクト体験](./04_git_team.md)

5. [【実践3】まるばつゲームを作ろう（Ruby・対コンピュータ）](./05_marubatsu_ruby.md)
